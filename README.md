meblog
======

to install meblog, do below please

1 clone the repo

    git clone git://github.com/Vinntoe/meblog.git
    
2 create directory db for store data

    cd meblog
    mkdir db

3 install the dependences

    npm install
    
4 start the mongoDB

    ./mongodb
    
5 start the app

    ./start

6 access http://localhost:3000 for meblog :)
